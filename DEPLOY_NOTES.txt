Render (backend)
Build Command: pip install -r requirements.txt
Start Command: uvicorn server:app --host 0.0.0.0 --port $PORT
Environment variable: PYTHON_VERSION=3.13.5

Vercel (frontend)
Environment variable name: VITE_API_URL
Environment variable value: https://YOUR-RENDER-SERVICE.onrender.com/api

Example:
VITE_API_URL=https://timetableg-backend.onrender.com/api
